# Code of Conduct

## 我们的承诺

为营造开放、友好的社区环境，我们承诺让每个人参与本项目与社区的体验不受骚扰，无论年龄、体型、可见或不可见的残障、族裔、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向。

## 行为准则

促进积极环境的行为包括：

- 表达同理与善意
- 尊重不同观点与经历
- 给予并优雅地接受建设性反馈
- 为错误承担责任并向受影响者致歉
- 以社区整体利益为先

不可接受的行为包括：

- 性化的语言或图像，及任何形式的性骚扰
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息（如住址、邮箱）
- 其他在专业场合中合理认定为不当的行为

## 执行

如遇滥用、骚扰或其他不可接受的行为，可通过仓库 Security 页或维护者公开的联系方式私下报告。所有投诉都会被及时、公正地审阅与处理；维护者有义务为报告者保密。

## 归属

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) v2.1。
